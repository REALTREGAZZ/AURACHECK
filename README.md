The ultimate offline Vibe Detector & NPC Scanner.

## Features
- **100% Offline**: No servers, no data collection.
- **AI Powered**: Uses `face-api.js` running locally on your device.
- **Vibe Stats**: Detects Sigma, Goofy, Dark, Lover, and NPC energy.
- **Shareable Cards**: Generates a meme-ready card of your scan.

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run locally:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```

## Models
AI models are located in `public/models`. They are downloaded automatically during setup.

## PWA
This app is a Progressive Web App. You can install it on your phone by adding it to the home screen.
